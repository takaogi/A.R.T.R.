# A.R.T.R. Roleplay Client

A.R.T.R. は、日本語での没入感の高いロールプレイ体験を目指して開発されたローカルLLMクライアントです。

## 特徴（Japanese Specialized）

本システムは**日本語のロールプレイに特化**しています。
RisuAI (.charx) 形式などの海外製キャラクターカードをインポートする際、プロンプト内で記述を**自動的に日本語へ翻訳・最適化**する機能を備えており、英語のキャラクターでもスムーズに日本語チャットを開始できます。

## 主な機能

- **自律思考AI**: ユーザーの入力を待たずに思考し、発言や行動を決定する能動的なAI。
- **没入型UI**: タイプライター風のテキスト表示、リアルタイムな立ち絵（表情差分）切り替え。
- **長期記憶**: 過去の会話や重要な出来事を記憶・想起するベクトルデータベース連携。
- **キャラクター管理**: RisuAI (.charx) 形式のインポートおよびネイティブ形式 (.artrcc) のサポート。
    - **注意**: インポートは「キャラクターカード」のみ対応しています。シナリオ形式やLorebook単体のインポートには対応していません。

## LLMプロバイダー

以下のプロバイダーに対応しています。

- **OpenRouter (推奨)**: 多数のモデルが利用可能で、安価かつ検閲の緩いモデルを選べるため、本システムの利用に最も適しています。
- **OpenAI**: GPT-5.2 / GPT-5-mini 等。

## 開発について

このプロジェクトは、素人が **Google Antigravity** の試運転として作成したものです。
コードや設計には未熟な点が含まれる可能性がありますが、AIエージェントによる開発の実験作として公開しています。
